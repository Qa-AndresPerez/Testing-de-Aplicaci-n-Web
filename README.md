# QA Testing Project – Web Application

## 📌 Descripción
Este proyecto consiste en la ejecución de pruebas manuales sobre una aplicación web con el objetivo de validar su correcto funcionamiento antes de producción.

## 🎯 Objetivo
Detectar errores en funcionalidades críticas y asegurar la calidad del software.

## 🔍 Alcance
- Pruebas funcionales
- Pruebas de regresión
- Validación de APIs

## 🧪 Actividades realizadas
- Creación de casos de prueba
- Ejecución de pruebas manuales
- Identificación y documentación de bugs
- Validación de endpoints con Postman

## 🛠 Herramientas utilizadas
- Postman
- MySQL
- Jira / Trello
- Navegadores (Chrome, Firefox, Edge)

## 📊 Resultados
Se identificaron errores en flujos críticos que fueron documentados y corregidos antes de su liberación.
